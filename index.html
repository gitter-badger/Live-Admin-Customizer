Live Admin Customizer.

Create Customized theme for wordpress admin panel simply selecting colors in theme creation panel And also manage created theme..

Plugin Feature List

  1. Built-in 5 Admin Themes
  2. Create New Themes
  3. Delete Created Existing Themes

Wordpress Installation

1. Download Latest Source From [Github Download](https://github.com/technofreaky/Live-Admin-Customizer/archive/master.zip) [Wordpress Download](https://wordpress.org/plugins/live-admin-customzier)
2. Activate the plugin through the \'Plugins\' menu in WordPress
3. Create New Theme Using [Live Admin Customizer]
